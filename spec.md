## 상세 변경

### 1. 히어로 서브카피 변경
- 기존: "그 사람도 같은 마음일까 / 대화에 답이 있을지도 몰라요"
- 변경: "카카오톡 대화를 넣으면 / 우리 사이 온도를 알려드려요"

### 2. 위스퍼 섹션 삭제
- `.empathy` 섹션 전체 제거 (HTML + CSS + scroll reveal delay)

### 3. 프리뷰 라벨 변경
- 기존: "이런 걸 알 수 있어요"
- 변경: "이런 결과를 받아볼 수 있어요"

### 4. 가이드 슬라이드 섹션 (신규)
- 위치: 프리뷰 카드 아래, CTA 존 위
- 라벨: "카톡 대화 내보내는 법"
- 슬라이드 4장 (guide/1.png ~ 4.png), 각 슬라이드에 짧은 설명:
  - 1: "채팅방에서 ⚙️ 설정을 눌러주세요"
  - 2: "스크롤을 내려 '대화 내용 내보내기'를 눌러주세요"
  - 3: "'모든 메시지 내부저장소에 저장'을 선택해주세요"
  - 4: "생성된 KakaoTalkChats.txt를 업로드하면 끝!"
- 인터랙션: **수동 스와이프만** (자동 넘김 없음)
- 하단 dot indicator (4개)
- 디자인: 기존 디자인 시스템 톤 유지 (다크 배경, 카드 surface 스타일)

### 5. 3스텝 필 제거
- `.how-steps` 제거 (가이드 슬라이드가 역할 대체)

## 삭제 대상
- `.empathy` 섹션 (HTML)
- `.whisper`, `.empathy-close` (CSS)
- `.empathy .whisper:nth-child` scroll reveal delay (CSS)
- `.how-steps`, `.how-pill`, `.pill-num`, `.pill-text` (HTML + CSS)

## 추가 대상
- `.guide` 섹션 (HTML)
- `.guide-slider`, `.guide-slide`, `.guide-dots` (CSS)
- 슬라이드 스와이프 JS (touch + mouse drag)

## 변경하지 않는 것
- 히어로 두 점 애니메이션, SCROLL 힌트
- 프리뷰 카드 내부 (온도, 스탯, 블러)
- CTA 버튼, 알림 신청 모달
- 프라이버시 라인 (UTM variant)
- 별 배경, 모든 기존 애니메이션
- Meta Pixel
